# TRACE

**TRACE** = **Textual Reuse, Alignment, and Collation Engine**

TRACE is a Python library for witness-based textual alignment, collation, and variant analysis.
It is designed for textual criticism, fragment comparison, and the creation of digital synopses and critical editions.

(c) Benjamin Schnabel, 2026

## Goals

- import witness data from JSON / TEI / related formats
- normalize and segment textual witnesses
- align witnesses using configurable scoring
- build master alignment structures
- support variant analysis, collation, and editorial workflows
- integrate with eScriptorium and related digital humanities tools
